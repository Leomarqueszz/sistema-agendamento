# Sistema de Agendamento 

## Sobre o projeto

Esse projeto foi desenvolvido para praticar Python e banco de dados através de uma situação comum do dia a dia: o agendamento de horários.

A ideia foi criar um sistema simples onde é possível cadastrar clientes em determinados horários, consultar os agendamentos já realizados e cancelar agendamentos quando necessário.

## Tecnologias

- Python
- SQLite

## Funcionalidades

- Realizar agendamentos
- Consultar agendamentos
- Cancelar agendamentos
- Validação para evitar horários duplicados

## Como pensei no projeto

Quando comecei o projeto, pensei em um problema simples que vi acontecer em uma barbearia aqui perto de casa. Muitas vezes os horários eram marcados manualmente, o que podia gerar confusão ou até mesmo dois clientes no mesmo horário. A partir dessa ideia, desenvolvi um sistema simples de agendamento. Apesar de ter pensado inicialmente em uma barbearia, o mesmo sistema também pode ser utilizado em clínicas, academias e outros negócios que trabalham com horários marcados.

Para resolver esse problema, criei uma tabela para armazenar o nome do cliente, a data e o horário do atendimento. Depois desenvolvi uma validação para impedir que duas pessoas fossem agendadas no mesmo horário, evitando conflitos na agenda.

O principal objetivo desse projeto foi praticar consultas em banco de dados, validações e a organização das informações utilizando Python e SQLite. Além disso, consegui exercitar bastante minha lógica de programação ao transformar uma situação do dia a dia em uma solução simples e funcional.
